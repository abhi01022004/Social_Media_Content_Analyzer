Social Media Content Analyzer (MERN)

This project extracts text from PDFs and images, performs OCR, gives engagement suggestions, and generates a short NLP-based description of the content.

Run the server:
  cd server
  npm install
  cp .env.example .env
  npm run dev

Run the client:
  cd client
  npm install
  npm run dev

To download face-api models for client-side face detection:
  node download-face-models.js
